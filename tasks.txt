01

Два из трёх
Напишите программу, которая выводит два из трёх введённых чисел.
На вход подаются три целых числа, записанные в одной строке через пробел. Каждое число в диапазоне [-100,+100].
На выход подайте в одной строке через пробел два любых числа из трёх введённых в любой последовательности.

04

Квадратная стрелка
На вход подаётся одной целое число n в диапазоне 1<=n<=100.
Число n обозначает размерность квадрата, для размещения диагональной стрелки - смотри примеры - пустоты заполняются пробелами. Для рисования используется символ 'X' - английская заглавная буква.
На выход на экран выводится в виде символов стрелка в квадрате.


05

Квадратный шифр
На вход подаются:
в первой строке одно число n размер квадрата 3<=n<=100
в последующих n строках символами X (это английская заглавная буква) означены границы квадрата и содержимое квадрата, а пустоты заполнены пробелами.
В квадрате символами X зашифровано одно число 0<=m<=64.
Чтобы расшифровать его достаточно просто подсчитать количество непустых клеток во внутренней части квадрата.
На выход на экран выводится одно расшифрованное число m.

07

Последняя цифра
Пусть в строке располагается ряд цифр.
Если их брать по одной слева-направо, то постепенно мы наберём всё множество цифр [0; 9] десятичной системы счисления. Нужно найти позицию, на которой набираемое множество станет полным. Пусть счёт цифр в строке начинается 1.
На вход подаётся строка, состоящая только из цифр (0 может стоять на любой позиции) длиной не более 255.
На выход нужно вывести номер искомой позиции или fail, если такой позиции нет (то есть множество цифр осталось неполным).


08

Настоящие боги
Древние греки за каждым проявлением сил природы видели деяние определённого бога. По мере освоения и изучения материального мира накопились сотни богов. Поклоняться и приносить подношения и жертвы такому количеству богов стало слишком накладно. Было принято решение провести ревизию и сократить список, убрав ненастоящих богов.  В список настоящих могли попасть только те боги, у которых в имени нет повторов букв.
Ваша задача подсчитать количество настоящих богов.
На вход подаются:
в первой строке целое число n [1; 300],
в последующих n строках имена богов (только заглавные буквы).
На выход подайте одно целое число - количество настоящих богов.
В списке может не оказаться ни одного имени настоящего бога.


09

Спрятанный пароль
На вход подаётся список из целых положительных чисел. В нём есть одно и только одно число, которое не повторяется, то есть встречается в списке только 1 раз - это и есть спрятанный пароль.
Напишите программу поиска этого пароля.
На вход подаётся:
в первой строке одно целое число n - длина списка чисел [1; 10^3],
во второй строке через пробел подаются n целых чисел в диапазоне [0; 108].
На выход подайте найденный пароль.

Особенное соревнование
В деревне Бестолковке каждый год проводят соревнование по программированию.
В соревнованиях по программированию участвуют N участников (1 ≤ N ≤ 10^3).
Есть всего три призовых места и именно в этом соревновании не важны способности участников, так как награды вручаются случайным образом. Сначала разыгрывается "Золото", потом "Серебро", а уж затем и "Бронза".
Сколько существует вариантов распределения комплекта из золотой, серебряной и бронзовой медалей?
Учтите, что одно призовое место может занять только один участник и участник не может взять сразу две медали.
Входные данные
На вход подаётся одно целое число N – количество участников.
Выходные данные
На выход выведите единственное число – искомое количество способов награждения участников.

Особенный лифт - 1
Лифт находится на первом этаже бесконечноэтажного здания. Особенность его в том, что для движения вверх у него есть только одна клавиша и каждое нажатие на неё приводит к движению вверх на N этажей, остановка (можно выйти), потом движение вниз на M этажей, остановка (можно выйти). После этого можно снова нажать на клавишу вверх и будет повторено движение вверх и вниз с остановками. Предположим стоит задача подняться выше этажа H. Напишите программу для подсчёта необходимого количества нажатий клавиши вверх.
На вход подаётся одна строка, в которой через пробел написаны целые числа - N M H (1 <= N, M, H <= 1000).
На выход подайте ответ - сколько нажатий клавиши вверх потребуется, чтобы подняться выше этажа H.
Если, при указанных значениях N и M выполнить задачу подъёма невозможно, то следует вывести NO.

15

Особенный лифт - 3
Особенность лифта в бесконечноэтажном доме не только в том, как он поднимается, но и в том как он движется вниз. Есть только одна клавиша для движения вниз. При нажатии на неё лифт случайным образом может проехать на один этаж вниз или на один этаж вверх. Сколько существует путей от этажа H до цокольного этажа (там паркинг), то есть этажа с номером 0, если изначально разрешено только K одноэтажных перемещений (то есть нужно сделать строго K перемещений, нельзя больше или меньше). Уточню, что, если перечислять сверху вниз, то после этажей 3, 2, 1 идёт 0, то есть от этажа 2, до цокольного этажа нужно проехать два перемещения вниз. Если лифт доезжает до цокольного этажа при оставшихся неизрасходованных перемещениях, то движение останавливается и такая попытка не засчитывается.
Например, если H = 2 и  K = 4, то ответ 2 (возможные варианты: вниз-вверх-вниз-вниз и вверх-вниз-вниз-вниз).
На вход подаётся одна строка, в которой через пробел записаны целые числа H и K  (1 <= H, K <= 20).
На выход подайте количество путей.
